## 📌 Project Overview
This repository serves as my Digital Literacy Portfolio, completed as part of the CSE0001 course[cite: 1, 6]. Assuming the role of a Student Digital Ambassador, the goal of this project is to create resources that help peers navigate the digital world safely, professionally, and responsibly. 

All work is divided into five module-specific tasks, which are documented in their respective folders in this repository.

## 📂 Module Summaries

### Task 1: Digital Literacy Awareness Infographic
A one-page visual resource designed to help college students understand what digital literacy means, highlighting safe internet practices and the importance of a professional online presence.
-**Folder:** `task-1-presentation/` 

### Task 2: Build Your Student Digital Portfolio
Establishing a foundational professional digital footprint. This task includes the setup of my academic and developer profiles across key platforms.
- **Folder:** `task-2-portfolio/` 
- **Platforms:** GitHub, LinkedIn, Kaggle

### Task 3: Explore Coding & Collaboration Platforms
Familiarization with online tools for studying, practicing, and collaborating. This includes completing an introductory Python coding challenge on HackerRank and building a data-collection quiz.
- **Folder:** `task-3-platforms/`

### Task 4: Draft a Professional Email & Etiquette Guide
Applying proper digital communication rules.This module includes drafting professional emails for academic extensions and internship inquiries, alongside a checklist for responsible social media use.
- **Folder:** `task-4-email-etiquette/` 

### Task 5: Cybercrime Awareness Case Study & Prevention Guide
Researching a realistic cybercrime scenario to produce an awareness resource. This includes a case study on a specific cyber threat and an actionable prevention checklist tailored for college students in India.
- **Folder:** `task-5-cybercrime/`
